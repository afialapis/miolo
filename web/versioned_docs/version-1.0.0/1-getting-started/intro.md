# Introducción a Miolo

¡Bienvenido a la documentación de **Miolo**!

Miolo es un servidor todo-en-uno basado en Koa. Esta documentación está diseñada para ayudarte a comenzar y explorar todas las capacidades del paquete.

## Estructura de la Documentación

- **Guías**: Para aprender a configurar y usar Miolo paso a paso.
- **API**: Referencia detallada de todas las clases y métodos disponibles.
